# weather-notifier

A simple weather notifier using Twilio API and including city, temperature in Â°C for night and day, weather description and cloudiness in %.

### How to use it?

1. Download my script.
2. Write your data instead of `..................` (phone number and Twilio data) and choose city in line 37.
3. Run the script in terminal -> `python weather-notifier.py`
